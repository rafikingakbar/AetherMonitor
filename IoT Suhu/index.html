Realtime IoT
Monitoring Dashboard

Proyek ini mencatat suhu dan kelembapan secara real-time menggunakan ESP32 dan sensor DHT, lalu dikirim ke server dan ditampilkan dalam dashboard web interaktif. Semua data dapat dianalisis secara statistik.

Jelajahi Sekarang

Fitur Utama

Pemantauan Suhu

Sensor DHT membaca suhu ruangan dan mengirimkannya ke server secara berkala.

Pemantauan Kelembapan

Kelembapan lingkungan juga dipantau dan ditampilkan secara real-time.

ESP32 + WiFi

Menggunakan ESP32 untuk mengirim data langsung melalui jaringan Wi-Fi.

Dashboard Interaktif

Web dashboard menampilkan data, analisis, dan grafik secara dinamis.
